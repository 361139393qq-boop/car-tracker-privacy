<h2>一、我们如何收集和使用您的信息</h2>
<p>本应用仅用于展示您自己的车辆位置信息，我们可能会收集并使用以下信息：</p>
<ul>
  <li>位置信息：包括车辆上传的经纬度、时间等，用于在地图上显示当前位置或轨迹。</li>
  <li>设备信息：如设备型号、操作系统版本、网络类型等，用于故障排查和统计分析。</li>
  <li>日志信息：如错误日志、访问时间等，用于提升应用稳定性。</li>
</ul>

<h2>二、信息的存储与保护</h2>
<ul>
  <li>位置信息仅用于在本应用中向您展示，不会用于广告或向无关第三方出售。</li>
  <li>我们会采取合理的安全措施保护您的数据安全，但由于技术限制，无法保证绝对安全。</li>
</ul>

<h2>三、第三方服务</h2>
<p>本应用集成了以下第三方服务：</p>
<ul>
  <li>高德地图，用于展示地图和位置。</li>
  <li>MQTT/云平台（如有），用于接收车辆设备上传的位置信息。</li>
</ul>
<p>这些第三方服务提供商可能会根据其自身的隐私政策收集和使用数据，建议您同时阅读其隐私政策。</p>

<h2>四、您的权利</h2>
<p>您有权：</p>
<ul>
  <li>查询、更正、删除与您相关的位置信息（在技术可行范围内）；</li>
  <li>撤回授权、停止使用本应用。</li>
</ul>
